# Справочник регионов

## Получение списка регионов

`GET /api/v1/region/get-all` вернёт список регионов.

### Запрос:

Параметры: отсутствуют.

### Ответ:

```json
{
  "items": [
    {
      "id": 3001,
      "name": "Россия, Москва"
    },
    {
      "id": 3002,
      "name": "Россия, Санкт-Петербург"
    }
  ]
}
```
