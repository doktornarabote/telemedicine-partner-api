# Справочник медицинских должностей

## Получение списка должностей

`GET /api/v1/medicalposition/get-all` вернёт список должностей.

### Запрос:

Параметры: отсутствуют.

### Ответ:

```json
[
  {
    "id": 1,
    "name": "Ассистент кафедры"
  },
  {
    "id": 2,
    "name": "Ведущий научный сотрудник"
  }
]
```
